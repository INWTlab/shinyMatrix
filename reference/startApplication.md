# Start Application

This function starts an example app from the folder \`inst\`.

## Usage

``` r
startApplication(app = "app", port = 4242)
```

## Arguments

- app:

  name of the folder in \`inst\`

- port:

  port of web application

## Examples

``` r
if (FALSE) { # \dontrun{
startApplication("appCopy")
} # }
```
